## Monty's Game 🎮

Simulação do Jogo de Monty Hall em C — com cores no terminal e sistema de replay.
*(Meu primeiro projeto na faculdade!)*

## Sobre o jogo

O Problema de Monty Hall é um clássico da teoria da probabilidade.
Você escolhe uma entre três portas. O apresentador abre uma porta com cabra.
Você decide: manter sua escolha ou trocar? 
Você pode ganhar uma cabra ou um prêmio em dinheiro.

## Recursos

- Cores ANSI no terminal (vermelho, verde, azul e amarelo)
- Sistema de replay
- Aleatoriedade baseada em horário
- Validação de entrada — o jogo rejeita portas inválidas
- Lógica de troca de porta fiel ao problema original

## Como compilar e rodar

```bash
gcc montysgame.c -o montys-game
./montys-game
```
